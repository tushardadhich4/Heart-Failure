# Heart-Failure
Heart Disease Prediction


This project implements a supervised machine learning model to predict the presence of heart disease in patients using 11 clinical input features: Age, Sex, ChestPainType, RestingBP, Cholesterol, FastingBS, RestingECG, MaxHR, ExerciseAngina, Oldpeak, and ST_Slope. The model outputs a binary prediction — HeartDisease = 1 (disease present) or HeartDisease = 0 (no disease)
